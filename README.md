# API de Análise de Dados

API REST desenvolvida em Python com FastAPI para simulação de processamento de dados e aplicação de regras de negócio.

## Funcionalidades

- Listagem de produtos
- Cadastro de produtos
- Cálculo automático de imposto
- Cálculo de preço final
- Análise de dados (total, média)

## Tecnologias

- Python
- FastAPI
- Uvicorn

## Como executar

```bash
pip install fastapi uvicorn
python -m uvicorn main:app --reload
